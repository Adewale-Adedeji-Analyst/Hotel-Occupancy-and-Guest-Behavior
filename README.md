# Hotel-Occupancy-and-Guest-Behavior
Power BI dashboard analyzing hotel occupancy trends, guest behavior, loyalty patterns, and booking # Hotel Occupancy & Guest Behavior Dashboard

## Project Overview

This Power BI dashboard was built to analyze hotel occupancy trends, guest booking behavior, loyalty segmentation, and reservation patterns within a hospitality environment.

The dashboard transforms raw booking and occupancy data into interactive visual insights that support operational planning, guest engagement strategies, and hospitality decision-making.

## Business Problem

Hotels manage large volumes of guest and reservation data daily, but without effective reporting and visualization, identifying occupancy trends and customer behavior patterns becomes difficult.

This dashboard was created to answer key hospitality and operational questions such as:

- How frequently are guests staying at the hotel?
- What booking patterns exist across weekdays and weekends?
- How far in advance do guests typically book?
- Which loyalty groups contribute most to occupancy?
- Which booking channels are most commonly used by different guest segments?

## Objectives

- Analyze hotel occupancy and stay patterns
- Understand guest booking behavior
- Evaluate loyalty membership distribution
- Compare booking channel performance
- Build an executive-style hospitality dashboard for operational insights

## Tools & Technologies Used

- Power BI
- Power Query
- DAX
- Excel
- Data Cleaning & Transformation
- Data Visualization

## Key Insights

- Most bookings were made less than one week before arrival
- Non-member guests represented the largest share of hotel stays
- Weekday occupancy remained relatively consistent throughout the period
- Loyalty segmentation revealed clear differences in booking behavior
- Booking channel usage varied across guest loyalty groups

- Booking lead-time analysis
